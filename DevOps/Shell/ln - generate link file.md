# ln

- 리눅스 파일 시스템에서 링크파일을 만드는 명령어
- 심볼릭링크 / 하드링크

## 심볼릭링크

- 단순히 원본 파일을 가리키도록 링크만 시켜둔 것(like 바로가기)

```sh
ln -s 몸통 바로가기

rm -rf 몸통 # 바로가기에 들어갈 수 없어진다.
```

## 하드링크

- 원본파일과 다른 이름으로 존재하는 동일한 파일이며 원본파과과 동일한 내용의 다른 파일이다.
- 서로 다른 파일이므로 둘 중 하나를 삭제하더라도 나무지는 그대로 남아 있음.
- mac os에서는 폴더는 작동하지 않는다.(폴더는 항상 심볼릭)

```sh
ln 몸통 제2몸통 # 둘은 연결되어 있으나 엄연히 다른 친구들.
```
