# 이상엽샘의 취미 선형대수학

선형대수학의 중심 축을 잡을 수 있도록 만드는 데에 주목

## 선형대수와 군 story

- ① matrix
  - matrix라는 대수구조 및 연산
  - 연립방정식
  - Row reduced echelon form
    - 기본행 연산
      - 연산을 행하다
      - 기본행렬의 곱셈
  - `AB = I <=> BA = I`
- ② vector space
  - vector space라는 대수구조 및 연산
  - subspace
  - 다양한 vector space의 예
  - isomorphism
- ③ basis and dimension
  - 일차결합, 생성
  - 선형독립, 선형종속
  - 기저
  - 기저를 만드는 방법
  - 차원
  - 우리의 철학
  - 다양한 차원의 예
- ④ linear map
  - linear map 정의, 특성
    - monomorphism, epimorphism, isomorphism과의 관계
  - 다양한 linear map의 예
  - dimension theorem
  - rank theorem
  - **linear extension theorem**
    - vector space의 분류
- ⑤ fundamental theorem of linear algebra
