# 관계와 분할

## 1. 관계

### 1-1. 용어의 정의

- 관계
  - 곱집합 `A x B`의 부분집합
  - `R = (A, B, P(x, y))`
    - A에서 B로의 관계
    - `P(x, y)`는 명제함수
- 관계 R의 해집합
  - `{(x, y)| A에 포함되는 x와, B에 포함되는 y에 대해서 P(x, y)는 참}`
  - e.g
    - `A = {2, 3}, B = {4, 6}`, `A x B = {(2,4), (2,6), (3,4), (3,6)}`
    - 여기서 `P(x, y)` x는 y의 약수
      - `R* = {(2,4), (2,6), (3,6)}`
      - `(2,4)는 R에 속한다 == 2R4`
- 정의역
  - 적당한 `y 포함 B` 대해서, `xRy`인 모든 `x 포함 A`의 집합 `Dom(R)`
- 상
  - 적당한 `x 포함 A`에 대하여 `xRy`인 모든 `y 포함 B`의 집합 `Im(R)`
