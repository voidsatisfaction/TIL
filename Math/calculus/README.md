# 미적분학(Calculus)

움직임의 현상을 다루는 수학

- 미적분학 이전의 수학이 다루던 범위
  - 정수론
  - 부피 계산
  - 도형의 넓이
- 미적분학 이후의 수학이 다루던 범위
  - 속도
  - 가속도
  - 힘
  - 압력
  - 변화율
  - 일
  - 끊임없이 변화하는 현상세계 대상을 다루는 과학과 미적분학이 결합해서 깊이 있는 과학을 탐구 가능하게 함
