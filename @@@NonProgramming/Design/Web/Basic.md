## Spacing and layout

### 1. Use whitespace

요소와 요소 사이에 공백을 두라.

요소의 그룹 사이에 공백을 두라.

웹 사이트의 section마다 공백을 두라

너무 과장하지 말라.

### 2. Define hierarchy

공백은 웹 사이트 요소들 사이의 보이지 않는 관계를 나타낸다.

How to?

- Define where you want your audience to look first

- Establish a flow that corresponds to your content's message

- Use whitespace to build that flow
